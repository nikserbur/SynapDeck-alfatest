# SynapDesk

Локальное зашифрованное рабочее пространство: заметки, таблицы, диаграммы, PDF и граф связей в одном окне. Данные хранятся на диске пользователя. Windows и macOS.

Альфа-версия. Лицензия: GPL-3.0-or-later.

## Установка

### Windows 10/11 (64-bit)

1. Откройте папку `windows/` и запустите установщик:
   - `nsis/SynapDesk_*-setup.exe`, или
   - `msi/SynapDesk_*_x64_en-US.msi`.
2. Если SmartScreen показывает предупреждение (сборка без подписи): «Подробнее» → «Выполнить в любом случае».
3. Требуется WebView2: на Windows 11 уже есть, на Windows 10 ставится автоматически.
4. Запуск из меню «Пуск».

### macOS 11+

1. Откройте `.dmg` из папки `macos/`, перетащите SynapDesk в «Программы».
2. Первый запуск: ПКМ по приложению → «Открыть» → «Открыть» (сборка без подписи Apple).

## Функционал

- Заметки: редактор с заголовками, списками, задачами; режим Markdown; файлы `.md` на диске (формат совместим с Obsidian).
- Wiki-ссылки: `[[заметка]]`, `[[sheet:…]]`, `[[diagram:…]]`.
- Таблицы (Univer): ячейки, формулы, ссылки на заметки и другие таблицы.
- Диаграммы: бесконечный холст, блоки, стрелки, подписи, вложения заметок/таблиц/картинок, экспорт CSV и PNG.
- PDF: просмотр из дерева и вставка в заметки.
- Дерево и папки, несколько пространств (workspaces), drag-and-drop, множественное выделение (Shift / Ctrl·⌘).
- Граф связей между заметками, таблицами, диаграммами, PDF и картинками.
- Поиск по содержимому (полнотекстовый индекс).
- Вкладки и split на два столбца с синхронизацией прокрутки.
- Озвучка заметок,pdf и экспорт в MP3.
- Облачная синхронизация между устройствами (E2EE, опционально).
- Два пароля на хранилище: основной открывает все записи, пароль-прикрытие — только помеченные для него (на случай, когда открыть приложение требуют под принуждением).
- AI (опционально): расстановка пунктуации и краткая выжимка текста через локальный шлюз.

## Отличия от Obsidian

- Таблицы с формулами встроены, а не через плагины.
- Диаграммы встроены и связываются с заметками и таблицами.
- Ссылки работают между заметками, таблицами и диаграммами, а не только между заметками.
- Граф строится по всем типам контента, не только по заметкам.
- Шифрование хранилища из коробки.
- E2EE-синхронизация без отдельной подписки.
- Заметки остаются `.md` на диске — их можно читать и править внешними редакторами.

## Как подойти к тестированию

Тестируйте в первую очередь концепцию, а не полировку. Приложение сырое, мелкие баги и шероховатости ожидаемы — отмечайте их, но главный вопрос другой: годится ли это как место для ресёрча, систематизации и обдумывания темы.

Смысл такой: прогнать через приложение реальную задачу целиком и проверить, удобно ли в нём хранить, перечитывать, обновлять и фиксировать выводы. Материал и отчёты даёт AI — в приложении и через Cursor/Claude; приложение это место, где вы решаете, что с ними делать.

Базовый сценарий (на нём это и обкатывалось — идеи, проработка концепции и задач диплома):

1. Начните с mind map: разложите тему на узлы.
2. Узлы разверните в заметки и таблицы, свяжите их ссылками `[[…]]`.
3. Положите PDF-источники в папку темы и ссылайтесь на примеры из них.
4. Ветвитесь и углубляйтесь: смежные подтемы — отдельные заметки, связанные с исходными.
5. Откройте папку хранилища в Cursor или Claude, попросите ресёрч по нужной ветке и положите результат в заметку. Формат `.md` тот же, интеграция бесшовная: в корне хранилища лежит `AGENTS.md`, по которому редактор понимает структуру и систему ссылок.
6. Вернитесь в приложение и оцените: легко ли найти и перечитать, обновить, дополнить связями; удобны ли для этого заметки и таблицы.

В фидбэке по этому сценарию важнее всего: где процесс буксует и чего не хватает, чтобы это стало рабочим местом для исследований, — а не только список визуальных недочётов.

## Что протестировать

Дерево и файлы:

- Создание, переименование, удаление заметок, таблиц, диаграмм, папок.
- Перенос элементов между папками и в другое пространство.
- Множественное выделение (Shift / Ctrl·⌘) и массовые перенос/удаление.
- Закрытие и повторный запуск: данные и вкладки на месте.

Контент:

- Таблицы: формулы, ссылки на заметку и другую таблицу.
- Диаграммы: блоки, стрелки, вложения, экспорт PNG и CSV.
- Вставка картинок из файла, из буфера, drag-and-drop.
- Переходы по `[[…]]`, `[[sheet:…]]`, `[[diagram:…]]`.
- Озвучка заметки и кнопка «Стоп» (останавливается сразу, в том числе на длинном абзаце).
- Экспорт заметки в PDF.

Синхронизация (если вошли в облако):

- Изменения появляются на втором устройстве.
- Перенос файла между папками не дублирует его и не возвращает в старое место.
- Удаление на одном устройстве удаляет на другом.
- Большой PDF при переносе между папками не загружается заново.

Поиск и навигация:

- Поиск по содержимому, граф, тёмная и светлая темы, горячие клавиши.

Полезнее всего отчёты по скорости дерева, переносам файлов и синхронизации между двумя устройствами — эти части недавно переработаны.

## Известные ограничения

- Сборка без подписи: SmartScreen (Windows) и Gatekeeper (macOS) показывают предупреждения.
- Диктовка: на macOS работает; на Windows встроенной нет — используйте системную диктовку Win+H, она печатает в активное поле, включая редактор.
- Нет версий для iOS и Android.
- Нет плагинов.

## Данные и приватность

По умолчанию данные не покидают компьютер. При включённой облачной синхронизации на сервер уходит только зашифрованный контент; пароль на сервер не передаётся, ключи остаются у пользователя. Восстановления пароля нет: без пароля данные расшифровать нельзя. На этапе альфы держите резервную копию папки хранилища.
