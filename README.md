## Установка

### Windows

1. Запустите `.exe` из папки `windows/`.
2. Если SmartScreen предупреждает об издателе: **Подробнее** → **Выполнить в любом случае** (альфа-сборка без подписи).
3. Откройте **SynapDesk** из меню Пуск.

Нужен [WebView2](https://developer.microsoft.com/microsoft-edge/webview2/) — на Windows 11 обычно уже есть.

### macOS

1. Откройте `.dmg` из папки `macos/`.
2. Перетащите **SynapDesk** в **Программы**.
3. При первом запуске: **ПКМ по приложению → Открыть → Открыть** (если macOS не доверяет разработчику).
