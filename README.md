# SynapDesk — альфа-тест

Папка для раздачи тестовых сборок друзьям и бета-тестерам.

| Папка | Платформа | Что положить сюда |
|-------|-----------|-------------------|
| [`windows/`](windows/) | Windows 10/11 (x64) | Установщик `.exe` или `.msi` из сборки Tauri |
| [`dmg/`](dmg/) | macOS (Apple Silicon или Intel) | Файл `.dmg` из сборки Tauri |

Подробные инструкции для тестеров — в `README.md` внутри каждой папки.

**Версия сборки в этом наборе:** 0.1.0 (альфа)
