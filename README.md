## Что это

Локальное приложение для заметок, таблиц, диаграмм и PDF: всё хранится у вас на диске и шифруется. Главное — совместная работа.

## Совместное редактирование

Делитесь заметкой, таблицей, диаграммой или целой папкой по email — с правами «Просмотр» или «Редактирование». Несколько человек правят документ одновременно и видят курсоры друг друга, как в Google Docs. Только приватно: контент шифруется end-to-end, сервер его не видит. И работает офлайн — правки без сети сливаются автоматически, без конфликтов версий.

Остальное на месте: Markdown-заметки (файлы `.md` на диске), таблицы с формулами, диаграммы и mind map, граф связей, поиск, тёмная и светлая темы.

## Установка

- Windows: откройте папку `windows/`, запустите `.exe` (при предупреждении SmartScreen — «Подробнее» → «Выполнить в любом случае»).
- macOS: откройте `.dmg` из `macos/`, перетащите в «Программы»; первый запуск — ПКМ по приложению → «Открыть».

## Приватность

Данные не покидают компьютер; при синхронизации на сервер уходит только зашифрованный контент, пароль не передаётся. Восстановления пароля нет — держите резервную копию папки хранилища.
